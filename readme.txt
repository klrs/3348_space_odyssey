3348: A Space Odyssey
Ryhmä 6. Made by Ville Järviranta, Mikael Elgström & Kalle Rissanen

#################################################################################

3348: A Space Odyssey tells a tale of Pan Bannister, an intergalactic trash hauler who gets dragged into
an adventure of his life when he gets abducted by aliens!

#################################################################################

COMMANDS:

help
move (room)
pick up (item)
hack (room)
inventory
equip (item)
use (object)
look room
examine (item)
For an example if you are in an elevator you can just
input the floor (move floor 1)
The use command can be used with some objects in your inventory and with some objects in the room.

Combat:
attack (enemy name)
use (list usables and amount)
use grenade
use charge
In combat you can just type "attack" to attack the first enemy.
If you want to repeat your last command in combat\nyou can type "p" to do this.\n\n

#################################################################################

DEPENDENCIES:

- Python 3.4.4 - https://www.python.org/download/releases/3.3.4
- MariaDB - https://downloads.mariadb.org/
- Connector/Python - https://dev.mysql.com/downloads/connector/python/

#################################################################################

GAME INSTALLATION

1. Make sure you have dependencies mentioned above installed!

2. Open MariaDB Command prompt and type mysql -u root -p < [full path to the sql file]

3. Input your MariaDB root password.

4. Now open the version_0-1.py file in IDLE.

5. Run the code by pressing F5.

6. Play the game.
